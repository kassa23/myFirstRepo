Modern Web Browsers

The modern web browser is a group of codes which is together perform
a series of tasks to display a web page on the screen.
The browser has an embedded engine sometimes called a “JavaScript virtual machine”.

Different engines have different “codenames”. For example:

  1. V8 – in Chrome, Opera and Edge.
  2. SpiderMonkey – in Firefox.

Component of web browser

  1. User interface
  2. Browser engine
  3. Rendering engine
  4. Data persistence
  5. Networking
  6. JavaScript interpreter
  7. UI Backend
